Url2Img 参数说明

作者 : 拳头巴掌 源码 : Github

首先要运行你的项目

$ npm install

$ node app

空白图片

同时设定宽高
http://url/{width}

设定宽高
http://url/{width}/{height}

随机背景色
http://url/{width}/{height}/r

透明图片
http://url/{width}/{height}/t

指定背景色
http://url/{width}/{height}/{bgColor}

指定背景色和文字色
http://url/{width}/{height}/{bgColor}/{fontColor}
